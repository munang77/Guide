11깔끔한 메시 구조 만들기 중급

애니메이션에 적합한 메시 구조(토폴로지)와, 고해상도 모델을 저해상도로 변환하는 방법.

왜 메시 구조가 중요한가?

팔꿈치를 구부릴 때 피부가 자연스럽게 접히려면, 그 관절 부분의 면들이 구부러지는 방향을 따라 배치되어야 합니다. 면의 배치 구조가 잘못되면 리깅(뼈대 연결) 시 메시가 찢어지거나 이상하게 변형됩니다.

좋은 메시 구조의 원칙

  • 사각형 면(Quad) 위주: 삼각형 면은 가능하면 피합니다. 내보낼 때 자동 변환하므로 모델링 중에는 사각형으로 유지.
  • 루프를 따라 엣지 배치: 관절, 눈, 입 주위 엣지가 그 방향을 따라 돌아가야 자연스러운 변형이 됩니다.
  • 너무 많은 엣지가 한 점에 모이면 안 됨: 5개 이상의 엣지가 한 점에 모인 곳을 폴(Pole)이라고 합니다. 폴이 많으면 표면이 찌그러져 보일 수 있으니 필요한 곳 외에는 최소화.
  • 폴리곤 크기 균일: 같은 메시 안에서 면 크기가 너무 크게 다르면 렌더링 이상이 생길 수 있습니다.

5개 이상 꼭짓점 면(Ngon) 피하기

5개 이상의 엣지로 둘러싸인 면을 Ngon이라고 합니다. 게임 엔진에서 삼각형으로 자동 변환될 때 예측 불가능하게 변형될 수 있습니다. 가능하면 4개 이하의 엣지를 가진 면으로 분할하세요. 단, 평평하고 변형되지 않는 부분(바닥, 뚜껑)은 예외적으로 허용됩니다.

리토폴로지 (Retopology)

리토폴로지는 복잡한 고해상도 메시를 기반으로 더 적고 깔끔한 면으로 다시 메시를 만드는 작업입니다. 조각(스컬프트)으로 디테일이 많은 형태를 만든 뒤, 그 위에 게임용 간단한 메시를 그리는 과정입니다.

  1. 스컬프트로 만든 고해상도 메시 위에 새 Plane(평면)을 추가합니다.
  2. 새 메시에 Shrinkwrap 모디파이어를 붙입니다. Target을 고해상도 메시로 설정. 이러면 새 메시가 고해상도 표면에 착 달라붙습니다.
  3. 에디트 모드에서 면을 추가하면서 깔끔한 구조로 다시 그립니다.
  4. 완성 후 Shrinkwrap 모디파이어를 Apply(적용)합니다.
  5. 고해상도 메시의 디테일을 노멀 맵으로 베이킹해서 저해상도에 적용합니다 (15챕터 참조).

LoopTools 애드온

Edit > Preferences > Add-ons에서 LoopTools를 활성화하면 에디트 모드에서 우클릭 메뉴에 추가됩니다:

  • Circle: 선택한 버텍스들을 완벽한 원 모양으로 정렬.
  • Relax: 선택한 버텍스들의 간격을 균일하게 이완.
  • Space: 선택한 버텍스들의 간격을 균등하게.

12스컬프트로 디테일 조각하기 중급

점토를 빚듯 3D 메시를 조각하는 모드. 자연스러운 형태, 주름, 갑옷 디테일에 사용합니다.

스컬프트 모드 진입

오브젝트 선택 후 상단 모드 드롭다운 > Sculpt Mode. 또는 상단 Sculpting 워크스페이스 탭.

스컬프트를 하려면 메시에 충분한 폴리곤(면)이 있어야 합니다. 폴리곤이 너무 적으면 브러시 효과가 제대로 나오지 않습니다. 먼저 Subdivision Surface 모디파이어를 Apply해서 면을 충분히 늘린 뒤 시작하세요.

브러시 기본 조작

  • F: 브러시 크기 조절 (드래그).
  • Shift+F: 브러시 강도 조절.
  • Ctrl 누른 채로 클릭: 효과 반전 (올리기 ↔ 파기).
  • X: X축 대칭 켜기. 한쪽만 조각하면 반대편도 자동으로.

주요 브러시

브러시효과언제 쓰나
Draw표면을 올리거나 파기 (Ctrl = 파기)기본 형태 만들기
Clay Strips점토 덩어리처럼 붙이기볼, 근육, 살집
Grab잡아당기기코, 귀 형태 잡기
Smooth (Shift 누름)울퉁불퉁한 표면 부드럽게언제든 Shift 누르면 사용
Flatten평탄하게 다듬기갑옷 판, 평면 부위
Crease날카로운 주름 만들기갑옷 엣지, 주름

Dyntopo (다이나믹 토폴로지)

스컬프트하는 곳에 자동으로 폴리곤을 추가해 주는 기능입니다. 디테일이 필요한 부위에 집중적으로 폴리곤이 생깁니다. Ctrl+D로 켜고 끕니다.

!!

Dyntopo를 켜면 기존 UV 정보, 버텍스 그룹, 모디파이어 설정이 삭제됩니다. 스컬프트 전용 메시에만 쓰고, UV나 리깅이 끝난 메시에는 쓰지 마세요.

Multiresolution 모디파이어

Dyntopo 대신 쓸 수 있는 방법입니다. 모디파이어를 붙이고 Subdivide 버튼을 눌러 레벨을 올리면, 낮은 레벨에서 큰 형태를 잡고 높은 레벨에서 세밀한 디테일을 추가할 수 있습니다. UV와 리깅이 유지됩니다.

스컬프트 작업 순서 (큰 형태 → 디테일)

처음부터 주름을 파지 마세요. 항상 큰 덩어리에서 작은 디테일로 내려갑니다.

  1. 실루엣: Grab·Draw로 전체 비율과 큰 덩어리부터. 멀리서 봐도 무엇인지 알 수 있게.
  2. 2차 형태: Clay Strips·Inflate로 근육, 볼, 살집 같은 중간 덩어리.
  3. 디테일: Crease·Draw Sharp로 주름, 흠집, 갑옷 엣지. 이때 Dyntopo나 Multires 레벨을 올림.
  4. 마감: Smooth로 어색한 곳 정리. 과하면 Mask로 보호하고 부분만.

대칭과 마스크 활용

  • X 대칭(X): 얼굴·갑옷처럼 좌우 같은 것은 켜고 한쪽만 작업. Y·Z 대칭도 가능.
  • Radial 대칭: 방패 문양, 바퀴처럼 방사형으로 반복되는 형태에 (Symmetry 패널에서 횟수 지정).
  • 마스크 반전(Ctrl+I): 보호할 곳 대신 작업할 곳만 빠르게 지정.
  • 마스크 지우기(Alt+M): 전체 마스크 해제.
  • Face Sets: 메시를 영역으로 나눠 관리 — 갑옷 조각별로 따로 스컬프트할 때.

게임 에셋으로 쓰려면 — 리토폴로지 예고

스컬프트 결과는 폴리곤이 수십만~수백만이라 그대로 게임에 못 씁니다. 흐름은 이렇습니다:

  1. 고폴리곤으로 스컬프트 (디테일 풍부한 원본).
  2. 그 위에 저폴리곤 메시를 새로 만듦 (리토폴로지 — 고급에서 자세히).
  3. 고폴리곤의 디테일을 베이킹으로 노멀 맵에 구워 저폴리곤에 입힘.
  4. 결과: 가벼운 메시 + 디테일해 보이는 표면. 게임 캐릭터의 표준 제작법입니다.
TIP

로블록스 캐릭터/소품 정도면 스컬프트를 가볍게만 쓰고(Multires 2~3레벨) 바로 노멀 맵을 베이킹하는 게 효율적입니다. 영화급 디테일은 게임에서 거의 보이지 않습니다.

13커브로 파이프/케이블 만들기 중급

수학적 곡선(커브)으로 파이프, 케이블, 경로를 메시보다 쉽게 만드는 방법.

커브란?

일반 메시는 점, 선, 면으로 이루어져 있습니다. 커브는 수학적으로 정의된 곡선으로, 컨트롤 포인트를 움직여 형태를 조정합니다. 파이프, 전선, 리본 같은 길쭉한 형태를 만들기에 적합합니다.

커브 추가

Shift+A > Curve > Bezier. 가장 일반적인 커브 타입입니다.

에디트 모드에서 컨트롤 포인트를 선택하고 이동하면 커브 형태가 바뀝니다. 각 포인트에는 핸들(손잡이)이 있어 곡선의 방향을 조절합니다.

커브에 두께 주기 (파이프 만들기)

  1. 커브를 선택하고 오른쪽 Properties 패널에서 커브 아이콘 클릭.
  2. Geometry 섹션 > Bevel > Round 선택.
  3. Depth 값을 올리면 커브에 원형 단면 두께가 생깁니다.
  4. Resolution 값으로 단면의 부드러움을 조절합니다.
  5. Fill Caps를 켜면 양쪽 끝이 막힙니다.
TIP

커브를 메시로 변환하려면 Object 메뉴 > Convert > Mesh. FBX로 내보내기 전에 변환이 필요합니다.

핸들 타입

에디트 모드에서 컨트롤 포인트 선택 후 V:

  • Auto: 자동으로 부드럽게. 처음엔 이걸 쓰세요.
  • Aligned: 두 핸들이 일직선. 부드러운 연결.
  • Free: 핸들을 각각 자유롭게 조절.
  • Vector: 직선 세그먼트. 각진 연결.

14노드로 재질 만들기 (Shader Editor) 중급

시각적 노드 방식으로 복잡한 재질을 만드는 방법. 금속, 유리, 발광 재질 등.

Shader Editor란?

재질을 그래프(노드) 방식으로 만드는 편집기입니다. 블록(노드)을 연결해서 색상, 텍스처, 반사 등을 조합합니다. 상단 Shading 워크스페이스에서 아래쪽 에디터가 Shader Editor입니다.

자주 쓰는 노드

노드 이름하는 일언제 쓰나
Principled BSDF기본 셰이더. 대부분의 재질을 표현금속, 플라스틱, 피부, 유리 등 거의 모든 재질
Emission빛을 발산하는 재질이펙트, 네온, 발광 오라, 빛나는 효과

Transparent BSDF완전 투명한 재질Mix Shader와 함께 반투명 효과에 사용
Image Texture이미지 파일을 텍스처로 사용UV로 이미지 텍스처 붙이기

Node Wrangler 애드온 단축키

Preferences에서 Node Wrangler를 활성화하면 사용 가능:

  • Ctrl+Shift+클릭: 해당 노드 출력을 Viewer에 연결해서 중간 결과 미리보기.
  • Ctrl+T: Image Texture에 자동으로 Texture Coordinate + Mapping 연결.
  • Ctrl+Shift+T: Color, Roughness, Normal 맵 파일을 한번에 불러오기.

레시피: 빛나는 발광 재질 (이펙트용)

  1. 재질 추가 > Shader Editor 열기.
  2. 기본 Principled BSDF 노드 옆에 Emission 노드 추가 (Shift+A > Shader > Emission).
  3. Emission Color를 원하는 색상으로 설정. Strength를 5~15로 올립니다.
  4. Mix Shader 노드 추가. Shader 입력 1에 Principled BSDF, 입력 2에 Emission 연결.
  5. Layer Weight 노드 추가. Fresnel 출력을 Mix Shader의 Fac에 연결. 외곽 부분만 발광하는 오라 효과 완성.

레시피: 금속 재질

  1. Principled BSDF: Metallic = 1.0.
  2. Base Color: 짙은 회색 (#909090 정도).
  3. Roughness: 0.15~0.35 (광택 금속). 0.7~0.9 (녹슨 금속).
  4. 더 자연스러운 질감이 필요하면 Noise Texture → ColorRamp → Roughness 연결. 무작위 스크래치 효과.

15텍스처 직접 그리기 + 베이킹 중급

3D 표면에 직접 그림 그리기 + 고해상도 모델의 디테일을 텍스처 이미지로 굽기.

텍스처 페인팅

상단 Texture Paint 워크스페이스. UV가 있어야 작동합니다. 3D 뷰포트에서 직접 모델 표면에 그림을 그릴 수 있습니다.

브러시효과
TexDraw기본 칠하기. 색상이나 텍스처 패턴 도포
Soften경계를 부드럽게 블러 처리
Smear픽셀을 끌어당겨 번지는 효과
Fill연결된 영역 단색으로 채우기
TIP

텍스처 페인트 후 Image Editor에서 Image > Save As로 저장해야 합니다. 저장하지 않으면 블렌더를 닫을 때 사라집니다.

베이킹 (Baking) 이란?

베이킹은 렌더링 결과를 이미지 파일로 저장하는 작업입니다. 주로 고해상도 메시의 노멀(면 방향), 그림자, 색상 정보를 저해상도 메시의 UV에 "굽는" 방식으로 사용합니다. 이렇게 하면 저해상도 메시에도 고해상도처럼 보이는 디테일을 줄 수 있습니다.

노멀 맵 베이킹 방법

노멀 맵은 면이 어느 방향을 향하는지를 색상으로 표현한 텍스처입니다. 고해상도의 울퉁불퉁한 표면을 저해상도 메시에 적용할 때 씁니다.

  1. Properties > Render에서 렌더 엔진을 Cycles로 변경. 베이킹은 Cycles 전용입니다.
  2. 저해상도 메시의 Shader Editor에서 Image Texture 노드 추가. 새 이미지 만들기(1024x1024 권장). 연결은 하지 않고 선택된 상태로 둡니다.
  3. 저해상도 메시 선택 → Shift+클릭으로 고해상도 메시 추가 선택 (고해상도가 마지막 선택이어야 함).
  4. Properties > Render > Bake > Bake Type: Normal.
  5. Selected to Active 켜기. Extrusion 값 0.05~0.1.
  6. Bake 버튼 클릭. 완료 후 Image Editor에서 저장.
  7. 저해상도 재질에서 Image Texture 노드에 저장한 이미지 연결 → Normal Map 노드 통과 → Principled BSDF의 Normal에 연결.

3점 조명 (기본 셋업)

피사체를 자연스럽게 보이게 하는 기본 조명 배치입니다:

  1. Key Light (주광원): 가장 밝은 빛. 피사체 좌상단 45도에 배치. 주 그림자 생성.
  2. Fill Light (보조광): Key Light 반대편. 세기는 Key의 1/3~1/2. 그림자를 너무 어둡지 않게.
  3. Rim Light (역광): 피사체 뒤에서. 외곽선 강조로 배경과 분리감.

HDRI 환경 조명

HDRI는 실제 환경을 360도로 촬영한 이미지입니다. 이것을 배경으로 설정하면 자연스러운 반사와 환경광이 생깁니다.

  1. 상단 Shading 워크스페이스에서 왼쪽 상단 드롭다운을 World로 변경.
  2. Use Nodes 클릭.
  3. Environment Texture 노드 추가 → Open으로 HDRI 이미지(.hdr) 불러오기.
TIP

무료 HDRI는 polyhaven.com에서 받을 수 있습니다. 2K~4K 해상도면 충분합니다.

EEVEE Bloom 설정

Emission(발광) 재질에 빛 번짐 효과를 주려면 EEVEE의 Bloom을 켜세요:

Properties > Render > Bloom 체크. Threshold: 0.8 (이 밝기 이상에서만 번짐 발생). Intensity: 0.05~0.1. Emission Strength가 높은 재질이 실제로 빛나는 것처럼 보입니다.

!!

블렌더 4.2부터는 EEVEE가 EEVEE Next로 바뀌어 Bloom이 별도 항목이 아니라 Compositor의 Glare로 처리됩니다. 버전에 따라 메뉴 위치가 다를 수 있습니다.

조명 다루는 실전 팁

  • 밝기 단위: Point/Spot/Area는 Power(W) 값으로 조절. 너무 어두우면 100, 1000처럼 크게 올려보세요.
  • 그림자 부드럽게: 광원의 Radius(크기)를 키우면 그림자 가장자리가 부드러워집니다. 0이면 칼같은 그림자.
  • 색온도: 광원 Color를 따뜻한 주황(실내/노을)이나 차가운 파랑(달빛/그늘)으로 — 분위기가 확 삽니다.
  • 광원도 보이게: 마법 구체 같은 건 광원만으론 안 보입니다. Emission 재질의 작은 구를 광원 위치에 같이 두세요.
  • 빠른 배치: 광원을 선택하고 시점을 원하는 각도로 맞춘 뒤, 광원을 그 방향으로 옮기면 미리보기(Rendered 뷰)로 즉시 확인.

이미지로 저장하기 (출력 설정)

  1. Output Properties(프린터 아이콘)에서 해상도 설정 — 보통 1920×1080.
  2. F12로 렌더링 시작. 렌더 창이 뜹니다.
  3. 렌더 창에서 Image > Save As로 PNG 저장. 배경 투명이 필요하면 Output에서 RGBA + PNG.
  4. 이펙트 프리뷰처럼 빠르게 보려면 EEVEE로 전환 후 F12 — 거의 즉시 나옵니다.
TIP

게임 이펙트 메시(검기, 마법진 등)의 홍보용 이미지를 만들 땐 EEVEE + Bloom + 검은 배경이면 충분합니다. Cycles는 캐릭터 포트폴리오처럼 사실적 품질이 필요할 때만 쓰세요.

17뼈대(리깅)와 웨이트 중급

메시에 뼈대를 넣어 움직이게 하는 과정. 캐릭터 걷기, 로봇 관절, 기계 장치 움직임에 필요합니다.

리깅이란?

리깅은 메시(겉모습)에 뼈대(Armature)를 넣고 연결하는 작업입니다. 마치 인형에 철사 뼈대를 넣는 것과 같습니다. 뼈대를 움직이면 연결된 메시가 따라서 변형됩니다.

Armature 만들기

  1. Shift+A > Armature. 뼈대 하나가 생깁니다.
  2. 에디트 모드(Tab)에서 뼈대 끝을 선택하고 E를 눌러 새 뼈를 추가합니다.
  3. 오른쪽 패널(N 키)에서 각 뼈의 이름을 정합니다. 좌우 대칭이면 끝에 .L(왼쪽) .R(오른쪽) 붙이기.

메시에 뼈대 연결하기

  1. 메시 선택 → Shift+클릭으로 Armature 추가 선택.
  2. Ctrl+P > With Automatic Weights. 자동으로 뼈대와 메시를 연결합니다.
  3. 포즈 모드(Ctrl+Tab)에서 뼈를 움직여 결과 확인.

웨이트 페인팅

웨이트는 각 뼈가 메시의 어느 버텍스를 얼마나 강하게 움직이는지를 나타냅니다. 자동 웨이트가 이상할 때 직접 수정합니다.

  • 메시 선택 후 모드를 Weight Paint로 변경.
  • 색상: 빨간색(1.0) = 100% 영향, 파란색(0) = 영향 없음.
  • 포즈 모드의 뼈를 클릭하면 그 뼈의 웨이트 분포가 보입니다.
  • Add 브러시로 웨이트 추가, Subtract로 제거, Smooth로 경계 부드럽게.

IK (Inverse Kinematics)

IK는 "역방향 운동학"입니다. 끝 뼈만 움직이면 연결된 뼈들이 자동으로 자연스러운 포즈를 잡습니다. 발끝을 고정한 상태에서 걷기 애니메이션 만들 때 유용합니다.

  1. 포즈 모드에서 IK를 적용할 끝 뼈 선택.
  2. Bone Constraints 탭(뼈 아이콘) > Add Constraint > Inverse Kinematics.
  3. Chain Length: 영향받을 위쪽 뼈 개수. 발이면 2(발목 + 무릎).
RBX

뼈대가 있는 FBX를 Studio에 가져오면 뼈대 구조가 그대로 들어옵니다. Studio의 AnimationEditor에서 이 뼈대를 움직여 애니메이션을 만들 수도 있습니다.

18애니메이션 만들기 중급

키프레임으로 움직임을 만들고, 게임용 클립으로 내보내는 방법.

키프레임이란?

키프레임은 특정 시간(프레임)에 오브젝트의 상태(위치, 회전, 크기)를 저장한 것입니다. 두 키프레임 사이는 블렌더가 자동으로 보간(중간값 계산)합니다.

키프레임 삽입

  • 원하는 프레임으로 이동 (타임라인 하단 숫자 또는 화살표 키).
  • 이동/회전/크기 조절 후 I > Location / Rotation / Scale 선택.
  • 또는 변환할 속성(예: Location X) 위에서 마우스 우클릭 > Insert Keyframe.

Dope Sheet

모든 채널의 키프레임을 한눈에 볼 수 있는 에디터입니다. 타임라인보다 더 자세한 편집이 가능합니다. 에디터 타입을 Dope Sheet로 변경. 키프레임을 드래그해서 이동, S로 간격 조절, X로 삭제.

Graph Editor

키프레임 사이의 보간 곡선을 편집합니다. 곡선 모양으로 움직임의 "느낌"이 결정됩니다. 예를 들어 직선이면 일정한 속도, S자 곡선이면 자연스러운 가속/감속이 됩니다.

NLA Editor (클립 관리)

NLA Editor는 여러 애니메이션 클립을 레이어로 쌓거나 조합하는 에디터입니다. Idle(대기), Walk(걷기), Attack(공격) 클립을 각각 만들어서 관리할 수 있습니다.

  1. Dope Sheet의 상단 드롭다운을 Action Editor로 변경.
  2. 각 애니메이션(Idle, Attack 등)마다 새 Action을 만들어 별도로 저장.
  3. 완성된 Action을 Push Down 버튼으로 NLA Editor의 스트립으로 변환.

Shape Keys (모프 타겟)

메시의 형태 변형을 키로 저장하는 기능입니다. 눈 깜빡임, 입 모양, 얼굴 표정 변화에 씁니다.

  1. Properties > Object Data (메시 아이콘) > Shape Keys > +. Basis가 생성됩니다.
  2. 다시 +를 눌러 새 Shape Key 추가 (이름: "Smile" 등).
  3. 에디트 모드에서 버텍스를 이동해 원하는 표정 형태 만들기.
  4. Value 슬라이더를 0~1로 조절해서 변형 정도 조절. 슬라이더에 키프레임도 가능.
RBX

Roblox Studio는 FBX의 애니메이션 클립을 Action 단위로 가져옵니다. Idle, Walk, Attack을 별도 Action으로 분리해서 내보내면 Studio에서 각각 독립적으로 사용할 수 있습니다.

19씬 정리하기 중급

오브젝트가 많아지면 정리가 필요합니다. 컬렉션으로 그룹 관리.

컬렉션이란?

컬렉션은 오브젝트를 그룹으로 묶는 폴더 같은 기능입니다. 아웃라이너에서 관리됩니다. 컬렉션 단위로 숨기거나, 렌더에서 제외하거나, 특정 레이어에서만 보이게 할 수 있습니다.

컬렉션 관리

  • 새 컬렉션: 아웃라이너 오른쪽 클릭 > New Collection.
  • 이동: 오브젝트 선택 후 M > 컬렉션 선택.
  • 숨기기/보이기: 아웃라이너에서 눈 아이콘 클릭.

이름 규칙 (추천)

# 종류_이름_버전
MESH_TowerBase_v02
RIG_TowerArm
MAT_Metal_Rusty
CAM_MainRender

일관된 이름 규칙을 쓰면 오브젝트가 100개가 넘어도 찾기 쉽습니다.

작업 저장 습관

  • Ctrl+S: 수시로 저장. 블렌더는 가끔 예고 없이 종료됩니다.
  • 버전 관리: 중요한 변경 전에는 다른 이름으로 저장. tower_v01.blend, tower_v02.blend.
  • 자동 저장: Preferences > Save & Load > Auto Save 켜기.

  • Bool Tool — 빠른 불리언: 두 오브젝트 선택 후 Ctrl+-로 즉시 빼기. 06챕터 Boolean을 단축키로.
