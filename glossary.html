용어 사전

가이드에 나오는 어려운 용어를 쉽게 풀었습니다. 본문에서 점선 밑줄 단어에 마우스를 올리거나 탭해도 뜻이 나옵니다.

공통 — 3D 기본
메시 Mesh
점과 선과 면으로 이루어진 3D 모델의 형태 그 자체. 캐릭터, 나무, 검 모두 메시입니다.
버텍스 Vertex
3D 모델을 이루는 점. 점(버텍스)들이 모여 선(엣지)이 되고, 선이 모여 면(페이스)이 됩니다.
폴리곤 Polygon
3D 모델을 이루는 면 조각. 많을수록 정교하고 부드럽지만 게임이 무거워집니다. "로우폴리"는 폴리곤이 적은 각진 스타일.
텍스처 Texture
3D 모델 표면에 입히는 그림(이미지). 옷에 프린트된 무늬처럼, 모델에 색과 디테일을 입힙니다.
UV
3D 모델의 표면을 2D 그림 위에 어떻게 펼쳐서 붙일지 정한 지도. 선물상자를 펼친 포장지 전개도와 같습니다. UV가 틀어지면 그림이 엉뚱한 곳에 붙습니다.
머티리얼 Material
표면의 재질. 나무인지 금속인지 유리인지 — 무엇으로 만들어진 느낌인지를 정합니다.
렌더링 Rendering
3D 장면을 계산해서 화면에 그림으로 그려내는 과정. "렌더가 무겁다" = 그려내는 계산이 오래 걸린다.
셰이더 Shader
표면이 빛을 받았을 때 어떻게 보일지 계산하는 규칙. 반짝일지, 거칠지, 투명할지를 결정합니다.
알파 Alpha
이미지의 투명도 정보. 알파 0은 완전 투명, 1은 완전 불투명. PNG는 알파를 저장할 수 있고 JPG는 못 합니다.
노멀 맵 Normal Map
실제 굴곡 없이도 표면이 울퉁불퉁해 보이게 속이는 특수 이미지. 보라색 계열로 보이는 그 이미지입니다.
FBX
3D 모델을 프로그램끼리 주고받을 때 쓰는 파일 형식. 블렌더에서 만든 모델을 .fbx로 내보내 Studio로 가져갑니다.
임포트 / 익스포트 Import / Export
임포트는 외부 파일을 프로그램 안으로 가져오기, 익스포트는 밖으로 내보내기.
파티클 Particle
불꽃, 연기, 반짝이처럼 작은 입자를 잔뜩 뿌려 만드는 효과.
LOD
Level of Detail. 멀리 있는 모델을 자동으로 단순한 버전으로 바꿔 성능을 아끼는 기술.
블렌더
모디파이어 Modifier
원본을 직접 건드리지 않고 효과를 입히는 기능. 거울 복제(Mirror), 둥글리기(Subdivision), 휘기(Bend) 등. 언제든 끄거나 지울 수 있습니다.
에디트 모드 Edit Mode
모델의 점/선/면을 직접 뜯어고치는 모드. Tab 키로 전환합니다.
토폴로지 Topology
메시의 면들이 어떤 구조로 짜여 있는지. 사각형 면 위주로 깔끔하게 짜여 있어야 변형과 애니메이션이 잘 됩니다.
스컬프트 Sculpt
찰흙 빚듯 브러시로 모델을 조각하는 작업 방식.
베이킹 Baking
복잡한 디테일이나 조명 계산 결과를 이미지 한 장으로 구워내 저장하는 것. 고폴리곤의 디테일을 노멀 맵으로 베이킹해 저폴리곤에 입힙니다.
리깅 Rigging
모델 안에 뼈대를 심어 움직일 수 있게 만드는 작업.
웨이트 Weight
뼈가 움직일 때 주변 살(메시)이 얼마나 따라 움직일지 정한 영향력 수치.
노드 Node
기능 상자들을 선으로 연결해 재질이나 효과를 조립하는 방식. 블렌더 셰이더와 Geometry Nodes가 이 방식입니다.
프로시저럴 Procedural
그림 파일 없이 수학 규칙으로 무늬와 형태를 자동 생성하는 방식. 무한히 확대해도 안 깨집니다.
컴포지터 Compositor
렌더링이 끝난 이미지를 후보정(색감, 글로우, 블러)하는 블렌더 내장 편집기.
OSL
Open Shading Language. 셰이더를 코드로 직접 짜는 언어. 초고급 영역입니다.
Luau · 스크립트
Luau
Roblox에서 쓰는 프로그래밍 언어. Lua를 개량한 것으로 문법이 짧고 배우기 쉽습니다.
변수 Variable
값을 담아 이름으로 꺼내 쓰는 상자. local coins = 10처럼 만듭니다.
함수 Function
자주 하는 작업을 묶어 이름 붙인 것. 이름만 부르면 그 작업이 실행됩니다.
테이블 Table
Luau의 만능 자료구조. 목록(배열)도, 이름표 달린 보관함(딕셔너리)도 전부 테이블입니다.
이벤트 Event
"무언가 닿으면", "플레이어가 들어오면"처럼 일이 생겼을 때 함수를 실행하도록 예약하는 것. Touched:Connect(...).
인스턴스 Instance
게임 안에 존재하는 모든 것 하나하나 (파트, 조명, 소리). 코드로 찾고 만들고 지웁니다.
서버 / 클라이언트 Server / Client
서버는 게임 규칙·점수를 관리하는 진실의 컴퓨터, 클라이언트는 각 플레이어의 화면. 점수 같은 진짜 값은 서버에서만 바꿔야 안전합니다.
Script / LocalScript
Script는 서버에서, LocalScript는 각 플레이어 기기에서 실행됩니다. 어디에 두는가가 어디서 실행되는가를 정합니다.
RemoteEvent
클라이언트와 서버가 대화하는 공식 통로. 클라가 서버에게 "이거 해줘"라고 부탁할 때 씁니다.
ModuleScript
여러 스크립트가 공유해 쓰는 코드 부품. require로 불러옵니다. 코드를 기능별로 정리하는 도구.
메타테이블 Metatable
테이블에 없는 키를 찾으면 어디서 대신 찾을지 정하는 장치. OOP(설계도로 객체 찍어내기)의 토대.
DataStore
게임을 껐다 켜도 데이터(코인, 레벨)가 남아있게 영구 저장하는 서버 기능.
CFrame
위치 + 바라보는 방향을 함께 담는 3D 좌표. 순간이동, 회전, 따라보기 등 3D 조작의 만능 도구.
Vector3
3D 공간의 한 점이나 방향 (x, y, z). 위치와 크기에 쓰입니다.
Humanoid
캐릭터 안의 "생명" 인스턴스. 체력, 이동속도, 점프, 사망을 담당합니다.
pcall
실패할 수 있는 코드(저장, 외부 호출)를 감싸 에러가 나도 전체가 죽지 않게 보호하는 함수.
익스플로잇 Exploit
해커가 클라이언트를 조작해 부정행위를 하는 것. 서버 검증으로 막습니다.
nil
"아무것도 없음"을 뜻하는 값. 빈 상자. "attempt to index nil" 에러는 없는 것을 건드렸다는 뜻.
Roblox Studio
퍼블리시 Publish
내가 만든 맵/게임을 Roblox 서버에 올려서 다른 사람들도 들어와 플레이할 수 있게 공개하는 것. File > Publish to Roblox.
파트 Part
Studio의 기본 블록. 상자, 공, 원기둥 등. 맵의 레고 조각입니다.
인스턴스 Instance
Studio 안에 존재하는 모든 것 하나하나. 파트, 조명, 소리, 폴더 전부 인스턴스이고 Explorer 창에 나열됩니다.
Explorer
씬에 있는 모든 인스턴스를 목록으로 보여주는 창. 맵의 차례표 같은 것.
Properties
선택한 것의 속성(크기, 색, 재질 등)을 보고 고치는 창.
에셋 Asset
모델, 그림, 소리처럼 게임을 만들 때 쓰는 재료 파일. Roblox에 업로드하면 고유 번호(Asset ID)가 생깁니다.
Toolbox
다른 사람들이 공유한 무료 모델/소리/이미지를 검색해서 가져다 쓰는 창고.
스폰 Spawn
캐릭터나 오브젝트가 게임 안에 나타나는 것. SpawnLocation은 플레이어가 시작하는 위치.
Anchored
파트를 그 자리에 고정하는 속성. 꺼져 있으면 중력으로 떨어집니다. 맵 파트는 전부 켜는 게 기본.
CanCollide
부딪힐 수 있는지 여부. 끄면 캐릭터가 그냥 통과합니다.
Union
여러 파트를 하나로 합치는 기능. Negate(빼기 모양)와 함께 쓰면 구멍을 뚫을 수 있습니다.
MeshPart
블렌더 같은 외부 프로그램에서 만든 메시를 담는 파트.
스터드 stud
Studio의 길이 단위. 캐릭터 키가 약 5 스터드.
스트리밍 StreamingEnabled
맵 전체가 아니라 플레이어 주변만 불러와서 메모리와 로딩 시간을 아끼는 기능. 큰 맵의 필수 설정.
스카이박스 Skybox
하늘처럼 보이게 맵 전체를 두르는 360도 배경 이미지.
Atmosphere
공기와 안개를 표현하는 기능. 멀리 있는 것이 뿌옇게 보이게 해 원근감을 만듭니다.
Raycast
보이지 않는 광선을 쏘아 무엇에 닿는지 검사하는 기능. 총알 판정, 바닥 감지 등에 쓰입니다.
Terrain
브러시로 산과 강을 빚는 Studio의 지형 시스템. 파트와 달리 자연스러운 곡면이 됩니다.
