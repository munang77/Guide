이 가이드의 특징

왜 이 가이드인가

FX

이펙트 레시피 7종

회오리 오라, 에너지 구체, 번개, 마법진, 파편, 연기, 홀로그램. 솔스RNG 스타일을 단계별로 분해해서 설명합니다.

RBX

Studio 연동

블렌더에서 만든 이펙트를 Studio의 ParticleEmitter, Beam, Trail로 재현하는 방법. Luau 코드도 포함.

0

이모지 없음

집중을 방해하는 이모지 없이, 텍스트 표시(TIP / !! / RBX / ADV)만 사용합니다.

PY

Python 자동화

반복 작업 자동화, FBX 일괄 내보내기, 오브젝트 배열 생성 등 실용 스크립트 포함.

KB

단축키 완전 정리

입문 치트시트부터 스컬프트, 애니메이션, NLA 에디터까지 카테고리별로 검색 가능한 표.

SVG

시각 다이어그램

화면 구성, 이펙트 구조, 파티클 흐름을 그림으로 설명. 글만으론 이해하기 어려운 부분을 시각화.

추천 학습 순서

어떻게 공부할까?

입문 -- 1~3주
기초 모델링 + FBX 내보내기

설치 → 화면 조작 → 간단한 타워 메시 제작 → UV → FBX. Studio에서 정상적으로 불러오는 것까지가 1단계 목표입니다.

중급 -- 2~4주
재질 노드 + 뼈대 + 애니메이션

Shader Editor로 금속/유리 재질 → 뼈대(Armature) 넣기 → Idle/Attack 애니메이션 제작. 움직이는 타워를 만드는 것이 목표.

고급 -- 3주+
이펙트 + 자동화

파티클/물리 이펙트 → 솔스RNG 스타일 오라/빔/마법진 제작 → Studio에서 재현 → Python으로 반복 작업 자동화.
