Luau 초급 — 코딩 첫걸음

코딩이 처음이어도 됩니다. 변수부터 자동문까지, Roblox 스크립트의 알파벳을 배우는 단계.

10챕터 변수와 조건문 함수 첫 실습

01스크립트란 무엇인가 초급

맵은 무대, 스크립트는 연출입니다. 문이 열리고 코인이 돌고 점수가 오르는 모든 "움직임"이 스크립트.

Luau란

Roblox에서 쓰는 프로그래밍 언어입니다. Lua라는 배우기 쉬운 언어를 Roblox가 개량한 것으로, 문법이 짧고 영어 문장처럼 읽힙니다. 코딩이 처음이어도 시작하기 좋은 언어입니다.

스크립트 만들고 실행하기

  1. Explorer에서 ServerScriptService 옆 + > Script 추가.
  2. 더블클릭하면 코드 편집기가 열립니다. 기본으로 적힌 코드가 보입니다.
  3. F5(Play)를 누르면 스크립트가 실행됩니다.
  4. View > Output 창을 열어두세요 — 스크립트가 말하는 내용과 에러가 여기 표시됩니다.

첫 코드

print("안녕, Roblox!")  -- Output 창에 글자를 출력

print는 괄호 안 내용을 Output 창에 적어주는 명령입니다. 앞으로 "코드가 어디까지 갔나" 확인하는 손전등처럼 계속 쓰게 됩니다.

TIP

-- 두 개 뒤의 글은 주석입니다. 컴퓨터는 무시하고 사람만 읽는 메모. 코드에 한국어로 메모를 달면서 배우세요.

02변수 — 이름 붙인 상자 초급

값을 담아두고 이름으로 꺼내 쓰는 상자. 모든 프로그래밍의 첫 단추.

local coins = 10            -- coins라는 상자에 10을 담음
local playerName = "문앙"   -- 글자(문자열)도 담을 수 있음
local isAlive = true        -- 참/거짓도 담을 수 있음

coins = coins + 5           -- 상자 내용을 바꿈 (이제 15)
print(coins)                -- 15

규칙

  • local을 붙여 만듭니다. 안 붙여도 되지만 붙이는 것이 표준이고 더 빠릅니다.
  • 이름은 영문으로, 뜻이 보이게: coins, maxHealth, isDoorOpen. 단어 경계는 대문자로 (camelCase).
  • 숫자로 시작 불가, 띄어쓰기 불가.

자료형 4총사

자료형예시용도
number10, 3.5, -2점수, 체력, 좌표
string"안녕"이름, 메시지. 따옴표로 감쌈
booleantrue / false켜짐/꺼짐, 살았나/죽었나
nilnil"아무것도 없음". 빈 상자

03연산자 — 계산과 비교 초급

더하고 빼고, 크고 작음을 비교하는 기호들.

계산

print(7 + 3)   -- 10
print(7 - 3)   -- 4
print(7 * 3)   -- 21 (곱하기)
print(7 / 2)   -- 3.5 (나누기)
print(7 % 3)   -- 1 (나눈 나머지)
print(2 ^ 3)   -- 8 (거듭제곱)
print("레벨 " .. 5)  -- "레벨 5" (..은 글자 이어붙이기)

비교 — 결과는 true/false

print(5 > 3)    -- true
print(5 <= 5)   -- true (작거나 같다)
print(5 == 3)   -- false (같은가? — 등호 두 개!)
print(5 ~= 3)   -- true (다른가?)
!!

= 하나는 "담아라", == 두 개는 "같은가?". 초보 에러 1위가 이 둘을 헷갈리는 것입니다.

논리 연산

-- and: 둘 다 참이어야 참 / or: 하나만 참이어도 참 / not: 뒤집기
print(true and false)  -- false
print(true or false)   -- true
print(not true)        -- false

04조건문 — 만약에 초급

"체력이 0이면 죽는다" — 상황에 따라 다르게 행동하게 만드는 문법.

local health = 30

if health <= 0 then
    print("죽었습니다")
elseif health < 50 then
    print("위험! 체력이 낮습니다")   -- 이게 출력됨
else
    print("건강합니다")
end
  • if 조건 then ... end가 뼈대. 조건이 true면 안쪽이 실행됩니다.
  • elseif로 추가 조건, else로 "그 외 전부". 위에서부터 검사해서 처음 맞는 곳 하나만 실행.
  • 안쪽 코드는 들여쓰기(Tab) — 문법상 필수는 아니지만 읽기 위해 반드시.

조건 조합

if hasKey and not isDoorOpen then
    print("문을 엽니다")
end

05반복문 — 여러 번 시키기 초급

같은 일을 100번 시키는 법. 코드를 100번 복붙하는 게 아닙니다.

for — 횟수를 알 때

for i = 1, 5 do
    print("카운트: " .. i)   -- 1 2 3 4 5
end

for i = 10, 2, -2 do      -- 10부터 2까지 2씩 감소
    print(i)                 -- 10 8 6 4 2
end

while — 조건이 참인 동안

local hp = 10
while hp > 0 do
    hp = hp - 3
    print("남은 체력: " .. hp)
end   -- 7, 4, 1, -2 출력 후 종료
!!

조건이 영원히 참이면 무한루프 — Studio가 멈춥니다. 반복 안에서 조건이 언젠가 거짓이 되는지 확인하고, 계속 도는 루프엔 task.wait()을 넣어 숨 쉴 틈을 주세요.

while true do           -- 게임 내내 도는 루프의 올바른 형태
    print("1초마다 실행")
    task.wait(1)
end

06함수 — 이름 붙인 작업 묶음 초급

자주 하는 일을 묶어 이름을 붙이고, 이름만 불러 실행합니다.

-- 정의: 이렇게 생긴 작업이다
local function greet(name)
    print("안녕, " .. name .. "!")
end

-- 호출: 실행해라
greet("문앙")    -- 안녕, 문앙!
greet("친구")    -- 안녕, 친구!

값을 돌려주는 함수 — return

local function add(a, b)
    return a + b          -- 결과를 돌려줌
end

local result = add(3, 4)  -- result에 7이 담김
  • 괄호 안 name, a, b는 매개변수 — 함수에 전달하는 재료.
  • return을 만나면 함수는 즉시 끝나고 값을 돌려줍니다.
  • 같은 코드를 두 번 복붙하고 있다면 그것이 함수를 만들 신호입니다.

07실습 — 파트 조작하기 초급

배운 것으로 진짜 게임 세계를 건드려 봅니다. 파트 색을 바꾸고 움직이게.

파트 찾아서 바꾸기

  1. Workspace에 Part를 하나 추가하고 이름을 MyPart로 바꿉니다.
  2. ServerScriptService에 Script를 추가하고 아래 입력:
local part = workspace.MyPart         -- 파트를 변수에 담기

part.BrickColor = BrickColor.new("Bright red")  -- 색 변경
part.Material = Enum.Material.Neon              -- 재질 변경
part.Transparency = 0.3                         -- 투명도
part.Size = Vector3.new(8, 2, 8)               -- 크기 (x,y,z)
part.Position = Vector3.new(0, 10, 0)          -- 위치

반복문과 조합 — 무지개 파트

local part = workspace.MyPart
while true do
    part.Color = Color3.fromHSV(math.random(), 1, 1)  -- 랜덤 색
    task.wait(0.5)
end

F5로 실행해 보세요. 0.5초마다 색이 바뀝니다. 점(.)으로 "Workspace 안의 MyPart의 Color"처럼 소속을 따라 내려가는 것 — 이것이 Roblox 스크립팅의 기본 동작입니다.

08에러 읽는 법 초급

에러는 혼내는 게 아니라 "여기가 문제야"라고 알려주는 안내문입니다. 읽을 줄 알면 무섭지 않습니다.

에러 메시지 해부

Workspace.Script:3: attempt to index nil with 'Color'
  • Workspace.Script — 어느 스크립트에서
  • :3: — 몇 번째 줄에서 (더블클릭하면 그 줄로 이동)
  • attempt to index nil — 무슨 문제 ("없는 것(nil)의 속성에 접근하려 함")

초급 에러 사전

에러뜻흔한 원인
attempt to index nil없는 것을 건드림이름 오타, 파트가 Workspace에 없음
is not a valid member of그런 자식/속성 없음철자, 대소문자 (position ≠ Position)
expected, got nil값이 와야 하는데 비어 있음변수에 담기 전 사용
'end' expectedend 짝이 안 맞음if/for/function의 end 누락
attempt to compare number with string숫자와 글자를 비교함"10"과 10은 다름 — tonumber() 필요
TIP

막히면 print를 곳곳에 심으세요. "여기까지 옴1", "여기까지 옴2" — 어디까지 출력되고 멈췄는지가 곧 범인의 위치입니다.

09종합 실습 — 자동문 초급

변수, 조건, 함수, 파트 조작을 전부 사용하는 첫 작품. 가까이 가면 열리는 문.

  1. Workspace에 Part 추가, 이름 Door, Anchored 켜기, 문 모양으로 크기 조절.
  2. ServerScriptService에 Script 추가:
local door = workspace.Door
local isOpen = false

local function openDoor()
    if isOpen then return end   -- 이미 열렸으면 무시
    isOpen = true
    door.Transparency = 0.8
    door.CanCollide = false      -- 통과 가능
    task.wait(2)                  -- 2초 후
    door.Transparency = 0
    door.CanCollide = true       -- 다시 닫힘
    isOpen = false
end

door.Touched:Connect(function(hit)   -- 닿으면 실행 (이벤트 맛보기)
    openDoor()
end)

마지막 두 줄이 이벤트 — "무언가 닿으면 이 함수를 실행해줘"라는 예약입니다. 이벤트는 중급에서 본격적으로 다룹니다.

10초급 치트시트 초급

초급 문법 한 장 요약. 막힐 때 돌아와서 보세요.

-- 변수
local x = 10
-- 조건
if x > 5 then ... elseif x > 0 then ... else ... end
-- 반복
for i = 1, 10 do ... end
while 조건 do ... task.wait(1) end
-- 함수
local function f(a, b) return a + b end
-- 파트 접근
local p = workspace.MyPart
p.Color = Color3.fromRGB(255, 0, 0)
p.Position = Vector3.new(0, 10, 0)
-- 기다리기
task.wait(1.5)
-- 이벤트 (맛보기)
part.Touched:Connect(function(hit) ... end)
TIP

여기까지가 "문법". 다음 단계(중급)부터 테이블, 이벤트, 클라이언트/서버 — 진짜 게임 기능을 만드는 도구들이 나옵니다.
