01설치와 화면 구성 입문

블렌더 다운로드부터 화면 구성 파악까지. 15분이면 됩니다.

다운로드

blender.org/download에서 최신 버전을 받습니다. Windows는 .msi 설치 파일, Mac은 .dmg를 받으면 됩니다.

TIP

버전 선택 화면에서 "LTS" 라고 적힌 버전을 고르세요. LTS는 오랫동안 안정적으로 관리되는 버전이라 처음 배우기에 좋습니다.

블렌더 화면 구성

꼭 해야 하는 초기 설정

  1. 단위 설정: 상단 메뉴 > Properties(오른쪽 패널) > Scene(카메라 아이콘) > Units > Length를 Centimeters로 변경. 로블록스 Studio 크기와 맞추기 위해 필요합니다.
  2. 자동 저장 켜기: Edit > Preferences > Save & Load > Auto Save 체크. 블렌더는 저장을 안 하면 작업이 날아갑니다.
  3. 넘패드 없을 때: 넘패드(숫자 키패드)가 없으면 Edit > Preferences > Input > Emulate Numpad를 켜세요. 키보드 위 숫자 1~9로 시점을 바꿀 수 있습니다.

자주 쓰는 애드온 켜기

애드온은 블렌더에 기능을 추가하는 플러그인입니다. Edit > Preferences > Add-ons 에서 검색해서 체크박스를 켜면 활성화됩니다.

  • Node Wrangler: 재질 노드 편집할 때 단축키를 대폭 늘려줌. 셰이딩 작업 필수.
  • LoopTools: 에디트 모드에서 버텍스(점)를 원형이나 직선으로 정렬해 줌.

화면 표시 방식 바꾸기

Z를 누르면 원형 메뉴가 나옵니다:

  • Solid (기본): 색이 채워진 형태. 일반 작업 시 사용.
  • Wireframe: 메시의 선만 보임. 뒤쪽도 투명하게 보임. Alt+Z로 빠르게 전환.
  • Material Preview: 재질/텍스처가 어떻게 보이는지 미리보기.
  • Rendered: 실제 렌더링 결과를 실시간으로 보기. 무겁지만 정확.

오브젝트를 잃어버렸을 때

작업하다 보면 오브젝트가 화면 밖으로 날아갈 때가 있습니다. 오른쪽 위 아웃라이너에서 해당 오브젝트 이름을 클릭한 뒤, 넘패드 .을 누르면 그 오브젝트가 화면 중앙에 옵니다.

TIP

넘패드 /를 누르면 선택한 오브젝트만 보이고 나머지는 숨겨집니다. 복잡한 씬에서 하나만 집중해서 볼 때 유용. 다시 누르면 원래대로 돌아옵니다.

노트북이라 넘패드가 없다면

넘패드 없는 키보드면 위쪽 숫자키가 안 먹힙니다. 두 가지 해결책:

  • 설정으로 해결: Edit > Preferences > Input > Emulate Numpad 켜기. 이제 위쪽 숫자키 1~9가 넘패드처럼 동작합니다. (단, 메시 편집의 1/2/3 버텍스 전환과 겹치니 주의)
  • 제스처로 해결: 화면 오른쪽 위의 작은 좌표축(빨강·초록·파랑 공)을 클릭하면 그 방향 시점으로 전환됩니다. 마우스만으로 정면·측면·윗면 보기 가능.
  • 메뉴로: 뷰포트 상단 View > Viewpoint 에 정면/측면/윗면이 다 있습니다.

마우스 휠/가운데 버튼이 불편하면

  • 트랙패드: Preferences > Input > Emulate 3 Button Mouse 켜면 Alt+왼쪽드래그로 회전할 수 있습니다.
  • 회전 중심: Preferences > Navigation > Orbit Around Selection 켜면, 선택한 오브젝트를 중심으로 화면이 돕니다. 안 켜면 엉뚱한 곳을 중심으로 돌아 멀미가 납니다 — 거의 필수 설정입니다.

03오브젝트 추가/이동/복제 입문

블렌더에서 오브젝트는 메시, 조명, 카메라 같은 씬의 구성 요소입니다. 메시는 점과 면으로 이루어진 3D 모델 자체를 말합니다.

오브젝트 추가하기

Shift+A를 누르면 추가 메뉴가 나옵니다:

종류설명예시
Mesh (메시)실제로 보이는 3D 모델. 게임 에셋의 본체.Cube(직육면체), Sphere(구), Cylinder(원기둥)
Light (조명)씬을 밝히는 빛. 렌더링 시 필요.Point(전구), Sun(태양), Spot(무대조명)
Camera (카메라)렌더링 시점을 정하는 카메라.기본 카메라 1개 있으면 됨
Empty (빈 오브젝트)실제로 보이지 않는 기준점. 그룹 중심, 경로 등에 사용.Plain Axes, Arrows

이동, 회전, 크기 조절

기능단축키축 제한 예시
이동GG → Z → 숫자 입력: Z축으로 정확하게 이동
회전RR → Z → 45 → Enter: Z축으로 45도 회전
크기SS → X → 2 → Enter: X축으로 2배 확대
TIP

G, R, S 를 누른 뒤 바로 숫자를 입력하면 정확한 값으로 변환됩니다. 예: GZ 누른 뒤 2 입력 후 Enter = Z방향으로 2 이동.

복제

  • Shift+D: 복제하고 바로 이동 모드. 우클릭하면 원래 위치에 복제됨.
  • Alt+D: 메시 데이터를 공유하는 복제. 하나를 수정하면 나머지도 같이 바뀜. 같은 모양의 오브젝트를 여러 개 배치할 때 메모리 절약.

오리진 (기준점)

오브젝트마다 기준점(오리진)이 있습니다. 주황색 점으로 표시됩니다. 회전하거나 크기를 바꿀 때 이 점을 중심으로 변환됩니다. 오른쪽 클릭 > Set Origin으로 위치를 바꿀 수 있습니다.

  • Origin to Geometry: 오브젝트 형태의 중앙으로 기준점 이동. 가장 자주 씁니다.
  • Origin to Cursor: 3D 커서(빨간 줄무늬 동그라미) 위치로 기준점 이동. 3D 커서는 Shift+우클릭으로 옮길 수 있고, Shift+C로 원점에 되돌립니다.

트랜스폼 적용 (중요!)

Ctrl+A > All Transforms 또는 Scale. 현재 이동/회전/크기 값을 "이게 기본값이다"라고 블렌더에 알려주는 작업입니다.

!!

FBX로 내보내기 전에 반드시 Scale(크기)을 Apply(적용)해야 합니다. 안 하면 Roblox Studio에 불러왔을 때 크기가 이상하게 들어옵니다.

선택, 숨기기, 삭제

  • 선택: 왼쪽 클릭. Shift+클릭으로 여러 개 선택.
  • 전체 선택: A. 다시 누르면 전체 해제.
  • 숨기기: H. 선택한 것만 숨김. Alt+H로 모두 다시 보이게.
  • 삭제: X 또는 Delete. 확인 창이 뜸 (실수 방지용).

실전 — 블록 위에 블록 올리기

가장 기본적인 작업입니다. 큐브 두 개를 정확히 위아래로 쌓아 봅니다. 이것만 손에 익으면 계단, 탑, 집 다 만들 수 있습니다.

  1. Shift+A > Mesh > Cube로 큐브 하나 추가. 기본 큐브는 한 변이 2m(원점이 중심이라 위로 1m, 아래로 1m)입니다.
  2. 다시 Shift+A > Cube로 두 번째 큐브 추가. 같은 자리에 겹쳐 생깁니다.
  3. 두 번째 큐브를 선택한 채로 G → Z → 2 → Enter. 위로 정확히 2m 올라가 첫 큐브 위에 딱 얹힙니다.
  4. 왜 2일까요? 두 큐브 모두 높이가 2m라, 한 칸 올리려면 정확히 2를 이동하면 됩니다. 같은 크기를 계속 쌓으려면 G Z 2를 반복하세요.
TIP

"왜 절반만 겹치지" "왜 공중에 떠 있지" 하는 혼란의 원인은 거의 다 원점이 중심에 있어서입니다. 큐브의 원점은 한가운데라, 바닥(Z=0)에 윗면을 맞추려면 큐브를 Z로 1만큼(높이의 절반) 올려야 바닥에 정확히 앉습니다.

실전 — 바닥에 정확히 앉히기

오브젝트가 바닥(빨간/초록 격자)을 뚫고 들어가거나 떠 있을 때, 바닥에 딱 맞추는 두 가지 방법:

  • 수치로: 높이의 절반만큼 Z로 올리기. 기본 큐브(높이 2m)면 G Z 1.
  • 스냅으로: 다른 오브젝트 위에 정확히 올리려면 스냅(자석)을 켜고 G로 끌어다 면에 붙이기 (다음 챕터에서 자세히).
  • N 패널 확인: N을 눌러 오른쪽에 뜨는 Item 패널에서 Location Z 값을 직접 입력해도 됩니다.

여러 개를 나란히/규칙적으로 놓기

  • 일렬 배치: 하나 만들고 Shift+D → X → 3 → Enter. X로 3칸 떨어진 복제본. 다시 Shift+D를 누르면 직전 간격을 기억해 같은 간격으로 또 생깁니다.
  • 격자 배치(Array 모디파이어): 똑같은 것을 줄줄이(울타리, 계단, 기둥) 만들 땐 손으로 복제하지 말고 Array 모디파이어 — 개수만 입력하면 자동 복제. (06 챕터)
  • 정렬: 여러 개를 선택하고 N 패널에서 같은 축 값을 입력하면 한 줄로 정렬됩니다.

04메시 직접 편집하기 입문

에디트 모드에서 메시의 점, 선, 면을 직접 편집합니다.

에디트 모드란?

오브젝트 모드는 오브젝트 전체를 선택하고 이동하는 모드입니다. 에디트 모드는 그 안에 들어가서 점(버텍스), 선(엣지), 면(페이스)을 직접 수정하는 모드입니다.

오브젝트를 선택하고 Tab을 누르면 에디트 모드로 들어갑니다. 다시 Tab을 누르면 나옵니다.

버텍스 / 엣지 / 페이스

이름설명전환 단축키
버텍스 (Vertex)메시를 구성하는 점1
엣지 (Edge)두 버텍스를 잇는 선2
페이스 (Face)여러 선으로 둘러싸인 면3

프로포셔널 편집

O로 켜고 끄는 기능입니다. 켜진 상태에서 점 하나를 이동하면 주변 점들도 함께 영향을 받습니다. 자연스럽게 구부리거나 불룩하게 만들 때 유용합니다. 마우스 휠로 영향 범위를 조절합니다.

정밀 이동 — 축 고정과 수치 입력

마우스로 대충 옮기지 말고, 정확한 방향과 거리로 옮기는 습관을 들이면 작업이 훨씬 빨라집니다. 이동(G)·회전(R)·크기(S) 모두 같은 방식으로 동작합니다.

입력뜻예시
G XX축으로만 이동좌우로만 움직임
G Z 2Z축으로 정확히 2만큼위로 2m 올림
G Shift+ZZ축 빼고 (XY 평면) 이동바닥에 붙인 채 이동
R Z 9 0Z축으로 90도 회전정확히 직각 회전
S 0크기를 0으로 — 한 평면에 정렬S Z 0 = 높이를 평평하게
이동 중 Shift천천히 미세 조정정밀하게 맞출 때
이동 중 Ctrl스냅 단위로 이동격자에 딱딱 맞춤
TIP

축을 두 번 누르면(예: G Z Z) 그 오브젝트가 기울어진 방향(로컬 축) 기준으로 이동합니다. 비스듬한 면을 따라 밀 때 유용합니다.

스냅 — 자석처럼 달라붙기

점·선·면을 다른 것에 정확히 붙이는 기능입니다. 윗줄 가운데 자석 아이콘으로 켜고 끄거나, 이동 중 Ctrl을 누르면 임시로 켜집니다.

  • 자석 아이콘 옆 드롭다운에서 스냅 대상 선택: Vertex(점에 붙기), Edge(선에 붙기), Face(면에 붙기), Increment(격자 단위).
  • Vertex 스냅이 가장 많이 쓰입니다 — 한 점을 다른 점에 정확히 겹칠 때.
  • Snap With 옵션으로 "선택의 어디를 기준으로 붙일지"(Center/Closest/Active) 지정.
  • 부품을 빈틈없이 이어 붙이거나, 바닥 격자에 정확히 올릴 때 필수입니다.

피벗과 기준점 바꾸기

회전·크기 조절이 "어디를 중심으로" 일어날지 정하는 설정입니다. 윗줄 가운데 점 두 개 아이콘(Pivot Point)에서 바꿉니다.

기준점중심쓰임
Median Point선택한 것들의 평균 위치기본값
3D Cursor3D 커서 위치특정 점 기준 회전 (.로 메뉴)
Individual Origins각자 자기 중심여러 개를 제자리에서 각각 회전
Active Element마지막 선택한 것한 점 기준 정렬
TIP

3D 커서를 원하는 곳에 놓으려면 Shift+우클릭. 원점으로 되돌리려면 Shift+C. 선택한 것 위로 커서를 보내려면 Shift+S > Cursor to Selected.

알아두면 편한 정리 기능

  • Merge by Distance: 거의 겹친 점들을 한 번에 합치기. 전체 선택 > M > By Distance. 임포트한 메시의 갈라진 점 정리에 필수.
  • Recalculate Normals: 면 방향 자동 정리 — Shift+N (다음 챕터에서 자세히).
  • Hide/Reveal: H로 선택한 것 숨기기, Alt+H로 전부 다시 보이기. 복잡한 안쪽을 작업할 때.
  • Checker Deselect: 한 칸 걸러 선택 — 루프 선택 후 Select > Checker Deselect. 규칙적인 패턴 작업에.
  • Select Similar (Shift+G): 비슷한 것 한 번에 선택 (같은 크기 면, 같은 방향 등).

05자주 쓰는 편집 도구 입문

에디트 모드에서 이것만 알면 기본적인 모델링이 됩니다.

돌출 (Extrude)

면, 선, 점을 바깥으로 밀어내서 새로운 부분을 만드는 기능. 블렌더에서 가장 많이 쓰는 도구입니다.

  • 면을 선택하고 E 누른 뒤 마우스 이동 또는 숫자 입력.
  • 예: 정육면체 위에 탑 만들기 = 윗면 선택 → E → Z → 2 → Enter

인셋 (Inset)

선택한 면 안쪽에 새 면을 만드는 기능. I를 누르고 드래그. 건물 창문, 패널 디테일, 구멍 뚫기 전처리에 씁니다.

루프컷 (Loop Cut)

메시 전체를 가로로 자르는 선을 추가합니다. Ctrl+R을 누르고 엣지 위에 마우스를 올리면 노란 선이 미리보기로 나옵니다. 클릭해서 확정하면 추가됩니다. 휠 스크롤로 개수를 늘릴 수 있습니다.

TIP

루프컷 추가 후 우클릭하면 중앙에 고정됩니다. 마우스를 움직이면 위치를 바꿀 수 있고, 좌클릭으로 확정합니다.

베벨 (Bevel)

날카로운 엣지를 둥글게 깎는 기능. Ctrl+B를 누르고 드래그. 마우스 휠로 단계(세그먼트) 수를 조절합니다. 단계가 많을수록 더 부드럽게 깎입니다.

나이프 (Knife)

메시에 자유롭게 선을 그어 자르는 기능. K를 누르고 클릭으로 선을 그립니다. Enter로 확정, Esc로 취소. 복잡한 면 분할에 씁니다.

채우기 (Fill)

열린 구멍을 면으로 메웁니다. 구멍 주변 엣지를 선택하고 F. 버텍스 두 개를 선택하고 F를 누르면 그 사이에 엣지가 생깁니다.

합치기 (Merge)

여러 버텍스(점)를 하나로 합칩니다. 버텍스를 선택하고 M. 합칠 위치를 선택합니다(At Center = 중간 지점, At Last = 마지막 선택한 점 위치).

분리 (Separate)

선택한 부분을 별도 오브젝트로 분리. P. 반대로 두 오브젝트를 하나로 합치려면 둘 다 선택 후 Ctrl+J.

노멀 (Normal) 방향

노멀은 면이 어느 쪽을 향하는지를 나타냅니다. 바깥쪽을 향하는 면은 게임에서 보이고, 안쪽을 향하는 면은 보이지 않습니다. 면 방향이 뒤집히면 게임에서 검게 보이거나 사라집니다.

  • 방향 확인: 뷰포트 오른쪽 위 Overlays 버튼 > Face Orientation. 파란색 = 정상, 빨간색 = 뒤집힘.
  • 자동 수정: 뒤집힌 면 선택(또는 전체 선택) 후 Shift+N. 바깥쪽 방향으로 자동 수정.
!!

FBX 내보내기 전에 반드시 Face Orientation을 확인하세요. 빨간 면이 있으면 Shift+N으로 수정해야 합니다. 게임에서 검은 면이 보이는 원인의 대부분이 이것입니다.

06모디파이어 입문

모디파이어는 원본 메시를 건드리지 않고 자동으로 변형을 적용하는 기능입니다.

모디파이어란?

예를 들어 Mirror 모디파이어를 붙이면 한쪽만 만들어도 반대쪽이 자동으로 생깁니다. Subdivision 모디파이어를 붙이면 메시가 자동으로 부드러워집니다. 원본은 그대로이므로 언제든 수정하거나 끌 수 있습니다.

추가 방법: Properties 패널(오른쪽 스패너 아이콘) > Add Modifier

입문 필수 모디파이어

이름하는 일언제 쓰나
MirrorX/Y/Z 축 기준으로 반대편을 자동으로 만들어 줌좌우 대칭 모델 만들 때. Clipping 옵션 켜면 가운데 틈이 생기지 않음
Subdivision Surface메시를 자동으로 부드럽게 세분각진 형태를 부드럽게. 뷰포트 레벨 2, 렌더 레벨 3 권장
Solidify얇은 면에 두께를 줌평면으로 만든 날개, 갑옷 판에 두께 추가
Bevel엣지를 자동으로 둥글게전체 오브젝트 엣지를 한번에 베벨
Array오브젝트를 일정 간격으로 여러 개 복제계단, 담장, 기어 이빨, 격자
Boolean두 오브젝트를 더하거나 빼거나 교집합구멍 뚫기(Difference), 형태 합치기(Union)
Decimate폴리곤 수를 줄여줌너무 많은 폴리곤을 최적화할 때. Ratio 0.5 = 절반으로
Simple DeformBend(구부리기)/Twist(비틀기) 등으로 형태 변형나선형 모양, 휘어진 형태

모디파이어 적용 (Apply)

모디파이어 결과를 실제 메시로 굳히는 것. 적용 버튼(v 체크 아이콘) 클릭. FBX 내보내기 시에는 Export 옵션에서 Apply Modifiers를 켜면 자동으로 적용됩니다.

!!

적용(Apply)하면 되돌리기(Ctrl+Z)를 저장 후에는 쓸 수 없습니다. 적용 전에 백업 파일을 저장해 두는 습관을 들이세요.

실전 1 — Array로 똑같은 것 줄줄이 만들기

계단, 울타리, 창살, 기둥처럼 같은 모양을 반복할 때. 손으로 복제하지 마세요.

  1. 반복할 오브젝트 하나 선택 > Add Modifier > Array.
  2. Count에 개수 입력 (예: 10 = 10개).
  3. Factor X/Y/Z로 간격 조절. Factor X = 1이면 자기 폭만큼 딱 붙어서, 1.2면 살짝 띄워서 배치됩니다.
  4. 계단을 만들려면 Factor X에 1, Factor Z에 1을 같이 주면 옆+위로 한 칸씩 = 대각선 계단.
  5. 마음에 들면 Apply. 원본 하나만 수정하면 전부 바뀌니, 수정이 끝난 뒤 Apply하세요.
TIP

Array를 두 개 겹치면 격자(가로 X Array + 세로 Y Array)가 됩니다. 바닥 타일, 벽돌 벽을 순식간에 만들 수 있습니다.

실전 2 — Mirror로 좌우 대칭 모델

  1. 오브젝트의 절반만 만들 준비를 하고 Add Modifier > Mirror.
  2. 기본은 X축 대칭. 한쪽을 만들면 반대쪽이 거울처럼 자동 생성됩니다.
  3. Clipping 켜기 — 가운데 경계에서 점들이 서로 통과하지 않게 막아, 중앙에 틈/겹침이 안 생깁니다.
  4. 캐릭터, 차, 갑옷처럼 좌우 같은 것은 작업량이 절반으로 줄어듭니다.

실전 3 — Boolean으로 구멍 뚫기

  1. 구멍 낼 본체와, 뚫을 모양의 오브젝트(예: 원기둥)를 겹쳐 놓습니다.
  2. 본체 선택 > Add Modifier > Boolean > Operation을 Difference(빼기)로.
  3. Object 칸에 뚫을 원기둥을 지정.
  4. 원기둥을 숨기거나(H) 다른 곳에 치우면 본체에 구멍이 뚫린 게 보입니다.
  5. Apply 후 뚫는 데 쓴 원기둥은 삭제. 창문, 문, 파이프 구멍에 유용합니다.
!!

Boolean은 가끔 면이 지저분해집니다(엉킨 토폴로지). 게임 에셋은 Boolean 후 정리(Merge by Distance, 면 다시 만들기)를 하거나, 단순한 형태는 에디트 모드에서 직접 뚫는 게 깔끔합니다.

모디파이어 순서가 중요합니다

모디파이어는 위에서 아래로 순서대로 적용됩니다. 순서가 바뀌면 결과가 완전히 달라집니다.

  • Mirror → Subdivision 순서: 먼저 대칭 만들고 부드럽게. (보통 이게 맞음)
  • Array → Bevel 순서: 먼저 복제하고 전체 엣지를 둥글게.
  • 순서는 모디파이어 목록에서 드래그(점 여섯 개 핸들)로 바꿉니다.
  • 일반 규칙: 형태를 만드는 것(Mirror, Array, Boolean)을 위에, 다듬는 것(Subdivision, Bevel)을 아래에.

07색상과 재질 입문

오브젝트에 색상, 광택, 금속 느낌 같은 재질을 적용하는 방법.

재질 추가

  1. 오브젝트 선택 후 오른쪽 Properties 패널에서 구체(Material) 아이콘 클릭.
  2. + 버튼 클릭 → New 클릭. Principled BSDF라는 재질이 생깁니다.
  3. Base Color의 색상 박스를 클릭해서 색을 고릅니다.

Principled BSDF 주요 설정

Principled BSDF는 블렌더의 기본 재질입니다. 현실의 거의 모든 재질을 표현할 수 있습니다.

항목범위설명
Base Color색상기본 색상. 텍스처를 연결하면 텍스처 색상이 사용됨
Metallic0 ~ 10은 플라스틱/천, 1은 금속. 중간값은 잘 안 씁니다
Roughness0 ~ 10은 거울처럼 완전 반사, 1은 무광. 금속은 0.2~0.4, 플라스틱은 0.5~0.8 권장
Alpha0 ~ 1투명도. 0은 완전 투명, 1은 불투명. Blend Mode를 Alpha Blend로 변경해야 작동
Emission Color색상발광 색상. 이펙트에 매우 중요. Emission Strength를 높이면 빛나는 느낌

한 오브젝트에 여러 색상

  1. Material 패널에서 +로 재질을 추가합니다(두 번째, 세 번째...).
  2. 에디트 모드(Tab)로 들어갑니다.
  3. 색칠할 면들을 선택합니다.
  4. Material 목록에서 적용할 재질을 클릭하고 Assign 버튼 클릭.
RBX

Roblox Studio에서는 SurfaceAppearance 오브젝트로 더 세밀한 재질을 표현합니다. 블렌더에서 만든 기본 재질은 FBX로 가져오면 색상 정도만 반영됩니다. 정밀한 텍스처는 8챕터 UV를 참조하세요.

08UV와 텍스처 입문

3D 모델의 표면에 이미지(텍스처)를 붙이는 방법.

UV란?

종이 박스를 펼치면 평평한 전개도가 되는 것처럼, 3D 메시를 2D로 펼친 것이 UV입니다. 펼쳐진 UV 위에 이미지를 올리면 3D 모델에 그림이 붙습니다. U와 V는 2D 좌표의 가로/세로 축 이름입니다(X, Y 대신).

Seam 표시하기

메시를 펼치기 전에 "어디를 잘라서 펼칠지" 경계선을 먼저 정해야 합니다. 이것이 Seam(봉제선 같은 개념)입니다.

  1. 에디트 모드(Tab)에서 엣지 선택 모드(2)로 변경.
  2. 잘릴 엣지를 선택합니다. 안 보이는 곳(뒤쪽, 밑면)을 고르세요.
  3. Ctrl+E > Mark Seam. 빨간 선으로 표시됩니다.

UV 펼치기 (Unwrap)

  1. 에디트 모드에서 전체 선택(A).
  2. U > Unwrap. 메시가 UV로 펼쳐집니다.
  3. 상단 탭에서 UV Editing 워크스페이스(작업 화면 구성 프리셋)로 이동하면 펼쳐진 모양을 볼 수 있습니다.
TIP

Seam 없이 빠르게 UV를 만들고 싶으면 U > Smart UV Project를 쓰세요. 자동으로 알아서 펼쳐줍니다. 각진 건물, 기계 부품에 특히 잘 맞습니다.

이미지 텍스처 연결하기

  1. 상단 Shading 워크스페이스로 이동.
  2. 아래쪽 노드 편집기에서 Shift+A > Texture > Image Texture 추가.
  3. Image Texture 노드에서 Open을 눌러 이미지 파일 선택.
  4. Image Texture의 Color 출력을 Principled BSDF의 Base Color 입력에 연결.

팔레트 텍스처 방식 (로블록스 권장)

로우폴리 게임 에셋에서 자주 쓰는 방식입니다. 여러 색상이 격자로 배치된 팔레트 이미지 하나를 만들고, UV를 그 위에 배치해서 색을 입힙니다.

  • 팔레트 이미지는 Photoshop, Aseprite, 또는 블렌더 Texture Paint로 직접 제작.
  • 해상도는 256x256 또는 512x512면 충분합니다. 그 이상은 불필요.
  • UV 에디터에서 각 Island(서로 연결된 채로 펼쳐진 면 덩어리)를 원하는 색상 영역 위로 옮깁니다. Island 위에 마우스를 두고 L을 누르면 통째로 선택됩니다.
TIP

UV는 게임 이펙트의 핵심이기도 합니다. 킹 레가시 같은 게임의 검기 이펙트가 바로 호 모양 메시 + UV 그라데이션 텍스처로 만들어집니다. 만드는 법은 블렌더 고급 강좌의 "게임 스킬 이펙트 메시" 챕터에서 다룹니다.

09FBX로 내보내기 입문

블렌더에서 만든 모델을 Roblox Studio에서 쓸 수 있도록 내보내는 방법.

내보내기 전 체크리스트

  • Scale 적용: Ctrl+A > Scale. 크기를 (1, 1, 1)로 초기화.
  • 면 방향 확인: Face Orientation으로 빨간 면 없는지 확인. 있으면 Shift+N으로 수정.
  • 이름 정리: 아웃라이너에서 오브젝트 이름을 알아보기 쉽게 바꾸기. 더블클릭하면 변경 가능.

FBX 내보내기 방법

  1. File > Export > FBX (.fbx) 클릭.
  2. 오른쪽 옵션 패널에서 아래 설정 확인.
  3. 저장 위치와 파일 이름 정하고 Export FBX 클릭.
