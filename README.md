# 블렌더 · Luau · Roblox Studio 완전 가이드

3D 모델링(블렌더), 게임 스크립트(Luau), 맵 제작(Roblox Studio)을 각각 입문부터 전문가까지 배우는 한국어 가이드 사이트입니다.

**사이트 바로가기 → https://munang77.github.io/Guide/**

---

## 세 개의 독립 코스 · 각 5단계

원하는 코스만, 원하는 순서로 봐도 됩니다. 세 코스는 서로 분리되어 있습니다.

### 블렌더 · 3D 모델링 (50+EX 챕터)

| 단계 | 페이지 | 내용 |
|---|---|---|
| 입문 | `beginner.html` | 설치, 모델링 기초, 재질, UV, FBX 내보내기 |
| 중급 | `intermediate.html` | 스컬프트, 셰이더 노드, 베이킹, 리깅, 애니메이션 |
| 고급 | `advanced.html` | 시뮬레이션, Geometry Nodes, Python + EX: 게임 스킬 이펙트 메시 |
| 초고급 | `master.html` | OSL 셰이더 코딩, 커스텀 애드온, 렌더팜, VFX 파이프라인 |
| 전문가 | `expert.html` | 프로덕션 파이프라인, 대규모 씬, 색 관리, 협업과 납품 |

### Luau · 게임 스크립트 (50 챕터)

| 단계 | 페이지 | 내용 |
|---|---|---|
| 초급 | `lua-beginner.html` | 변수, 조건, 반복, 함수 — 코딩 첫걸음 |
| 중급 | `lua-intermediate.html` | 테이블, 이벤트, 서버/클라이언트, GUI, Tween |
| 고급 | `lua-advanced.html` | 모듈, OOP, DataStore, CFrame, Raycast |
| 초고급 | `lua-master.html` | 아키텍처, 보안, 네트워크 최적화, 프로파일링 |
| 전문가 | `lua-expert.html` | ECS, 커스텀 리플리케이션, 서버 권위, 라이브 운영 |

### Roblox Studio · 맵 제작 (50 챕터)

| 단계 | 페이지 | 내용 |
|---|---|---|
| 초급 | `studio-beginner.html` | 설치, 파트, 지형, Toolbox, 머티리얼, 퍼블리시 |
| 중급 | `studio-intermediate.html` | 맵 설계, 건축 기법, 임포트, 조명, 사운드 |
| 고급 | `studio-advanced.html` | 대형 맵, 스트리밍, 최적화, 테마 연출, 모바일 |
| 초고급 | `studio-master.html` | 모듈러 키트, 환경 스토리텔링, 동선과 페이싱, 사운드스케이프 |
| 전문가 | `studio-expert.html` | 아트 디렉션, 성능 예산, 팀 협업, 라이브 맵 운영 |

### 부가 페이지

- `index.html` — 메인 허브
- `glossary.html` — 용어 사전 (검색 가능). 본문에서도 점선 밑줄 용어에 마우스를 올리면 뜻이 뜹니다.
- `style.css` — 공통 스타일

## 특징

- 사전 지식 없이 설치부터 따라올 수 있는 구성
- 세 코스 모두 전문가 단계까지 — 프로덕션 파이프라인, 라이브 운영, 상업 제작을 실제 깊이로
- 어려운 용어는 본문 툴팁 + 용어 사전으로 즉석 설명
- 추천 설정값과 실제 코드를 표·예제로 정리

## 직접 띄우기

이 저장소를 fork 한 뒤 Settings > Pages > Deploy from a branch (main, root) 로 설정하면 본인 계정에서도 사이트가 열립니다. 모든 파일이 같은 폴더(root)에 있어야 페이지 간 링크가 작동합니다.
