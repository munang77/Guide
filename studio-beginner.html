Studio 초급 — 맵 제작 기초

설치 첫날부터 첫 맵 퍼블리시까지. 파트, 지형, 머티리얼 — 맵 제작의 기본기를 잡는 단계.

01설치와 화면 구성 초급

Roblox Studio를 설치하고 각 창이 무슨 역할인지 파악합니다.

설치

  1. create.roblox.com 접속 > 로그인 > 우측 상단 Studio 다운로드.
  2. 설치 후 실행하면 템플릿 선택 화면이 나옵니다.
  3. Baseplate 템플릿으로 시작하세요. 평평한 바닥만 있는 빈 월드입니다.

핵심 창 4가지

창역할여는 법
Explorer월드 안 모든 오브젝트의 목록. 폴더 구조처럼 표시View > Explorer
Properties선택한 오브젝트의 속성(크기, 색, 재질 등) 편집View > Properties
Output스크립트 출력과 에러 메시지 표시View > Output
Toolbox다른 사람이 만든 무료 에셋 검색/사용View > Toolbox
TIP

Explorer와 Properties는 항상 켜두세요. 이 두 창 없이는 Studio 작업이 사실상 불가능합니다.

Explorer에서 꼭 알아야 할 서비스

  • Workspace: 게임 화면에 실제로 보이는 모든 것. 파트, 모델, 지형.
  • Lighting: 전체 월드의 조명, 하늘, 안개 설정.
  • StarterGui: 플레이어 화면에 표시되는 UI가 들어가는 곳.
  • StarterPlayer: 플레이어 캐릭터 관련 설정(이동 속도, 카메라 등).
  • ServerScriptService: 서버에서 돌아가는 스크립트를 두는 곳.
  • ReplicatedStorage: 서버와 클라이언트가 함께 쓰는 저장소.

02카메라 조작 초급

에디터 안에서 시점을 자유롭게 움직이는 법. 손에 익을 때까지 연습하세요.

조작동작
우클릭 + 드래그시점 회전
W A S D전후좌우 이동 (우클릭 누른 상태)
E / Q위로 / 아래로 이동
Shift 누른 채 이동느리게 이동 (정밀 작업)
마우스 휠줌 인/아웃
F선택한 오브젝트로 카메라 포커스
휠 클릭 + 드래그화면 평행 이동(팬)
TIP

오브젝트를 잃어버렸을 때는 Explorer에서 그 오브젝트를 클릭하고 F를 누르면 카메라가 바로 날아갑니다. 가장 많이 쓰는 키입니다.

카메라 속도 조절

File > Studio Settings > Camera에서 Camera Speed를 조절할 수 있습니다. 큰 맵 작업 시 속도를 올리면 편합니다. 우클릭 이동 중 마우스 휠을 굴려도 즉석에서 속도가 바뀝니다.

03파트 추가와 변형 초급

파트(Part)는 Studio 월드의 기본 블록입니다. 모든 건물, 지형, 구조물의 재료.

파트 추가

  1. Home 탭 또는 Model 탭 > Part 버튼 옆 화살표.
  2. Block(직육면체), Sphere(구), Cylinder(원기둥), Wedge(쐐기), Corner Wedge 중 선택.
  3. 카메라가 보고 있는 위치에 파트가 생성됩니다.

변형 도구 (이동/크기/회전)

도구단축키설명
Select1선택만. 드래그로 자유 이동
Move2축 화살표를 잡고 한 방향으로만 이동
Scale3구체 핸들을 잡고 크기 변경
Rotate4원형 핸들을 잡고 회전

스냅(Snap) 설정

Model 탭의 Snap to Grid 영역에서 이동/회전 간격을 정합니다:

  • Move 1 stud: 파트가 1 stud 단위로 딱딱 맞게 이동. 건축에 필수.
  • Rotate 45도: 회전이 45도 단위로 끊어짐.
  • 체크를 끄면 자유롭게 움직이지만 파트 사이에 미세한 틈이 생기기 쉽습니다.

복제와 정렬

  • Ctrl+D: 제자리 복제. 복제 직후 이동 도구로 옮기세요.
  • Alt+드래그: 이동하면서 복제. 벽, 울타리처럼 반복 배치에 최고.
  • Ctrl+Z / Ctrl+Y: 되돌리기 / 다시 실행.
!!

파트를 추가하면 기본으로 Anchored가 꺼져 있습니다. Play를 누르면 물리가 적용돼 파트가 굴러떨어집니다. 움직이면 안 되는 파트는 Properties에서 Anchored를 켜세요.

04파트 속성 초급

Properties 창에서 파트의 모든 성질을 제어합니다. 자주 쓰는 속성만 추리면 이렇습니다.

속성역할예시
Anchored물리 고정. 켜면 절대 안 움직임바닥, 벽, 건물: 항상 켜기
CanCollide충돌 여부. 끄면 통과 가능장식, 유령 벽: 끄기
Transparency투명도. 0(불투명)~1(완전투명)유리창: 0.5
Color파트 색상클릭해서 팔레트에서 선택
Material재질. 표면 질감과 반사Wood, Metal, Neon, Glass 등
Size크기 (X, Y, Z)바닥: 100, 1, 100
Position위치 좌표정확한 배치 시 직접 입력
Reflectance반사율 0~1거울 느낌: 0.5+
CastShadow그림자 생성 여부작은 장식: 끄면 성능 향상

재질(Material) 추천

  • SmoothPlastic: 깔끔한 기본. 로우폴리 스타일에 잘 맞음.
  • Neon: 스스로 빛나는 재질. 간판, 라이트, 이펙트에 사용.
  • Glass: 진짜 유리처럼 투과/굴절. Transparency와 함께 사용.
  • Wood / WoodPlanks: 나무 질감.
  • Brick / Concrete / Slate: 건물 외벽.
TIP

여러 파트를 선택하고 속성을 바꾸면 한꺼번에 적용됩니다. Ctrl+클릭으로 여러 개 선택하세요.

05모델과 Explorer 정리 초급

파트가 수십 개를 넘어가면 정리가 생명입니다. Model과 Folder로 구조를 잡으세요.

Model로 묶기

  1. 묶고 싶은 파트들을 Ctrl+클릭 또는 드래그로 선택.
  2. Ctrl+G로 Group(Model 생성).
  3. Explorer에서 Model 이름을 더블클릭해 의미 있는 이름으로 변경 (예: House, Tree).
  4. 풀려면 Ctrl+U (Ungroup).

Model은 하나처럼 선택/이동/복제할 수 있습니다. 집을 한 번 만들고 Model로 묶으면, Ctrl+D 한 번으로 집 전체를 복제할 수 있습니다.

Folder로 분류

Explorer에서 Workspace 옆 + > Folder. 폴더는 게임 동작에 영향 없이 정리만 담당합니다.

-- 추천 Workspace 구조
Workspace
  Buildings   -- 건물 모델들
  Nature      -- 나무, 바위
  Roads       -- 길 파트들
  Interactive -- 버튼, 문 등 스크립트 붙는 것들
!!

이름을 Part, Model 그대로 두면 나중에 수백 개 중에서 찾을 수 없습니다. 만들 때마다 바로 이름을 바꾸는 습관을 들이세요.

06지형 에디터 초급

산, 강, 동굴 같은 자연 지형은 Terrain 에디터로 만듭니다.

열기

Home 탭 > Editor (Terrain 영역). 왼쪽에 Terrain Editor 패널이 열립니다.

주요 도구

도구기능
Generate파라미터 기반 랜덤 지형 자동 생성. Biome으로 산/사막/설원 선택
Add브러시로 지형 추가. 언덕, 산 쌓기
Subtract지형 깎기. 동굴, 구덩이, 강바닥
Paint모양은 그대로, 재질만 변경 (잔디 → 돌)
Smooth각진 지형을 부드럽게
Flatten지정한 높이로 평탄화. 건물 부지 만들기
Sea Level기준 높이까지 물 채우기. 바다, 호수

작업 흐름 추천

  1. Generate로 대략적인 지형 생성 (또는 Add로 직접 쌓기).
  2. Flatten으로 건물 들어갈 자리 평탄화.
  3. Paint로 길, 절벽 등 재질 구분.
  4. Smooth로 어색한 경계 다듬기.
  5. Sea Level로 물 추가.
TIP

브러시 크기(Size)와 강도(Strength)를 조절하면서 작업하세요. 큰 브러시로 큰 형태를 먼저 잡고, 작은 브러시로 디테일을 넣는 순서가 효율적입니다.

07Toolbox와 에셋 초급

모든 걸 직접 만들 필요는 없습니다. Toolbox에서 무료 에셋을 가져다 쓸 수 있습니다.

Toolbox 사용법

  1. View > Toolbox.
  2. Models 탭에서 검색 (영어 검색이 결과가 더 많습니다. 예: tree, house, car).
  3. 클릭하면 월드에 추가됩니다.

주의할 점

  • 스크립트 확인: Toolbox 모델 안에 악성 스크립트가 숨어 있을 수 있습니다. 가져온 직후 Explorer에서 모델을 펼쳐 Script가 있는지 확인하고, 모르는 스크립트는 삭제하세요.
  • 파트 수 확인: 어떤 모델은 수천 개 파트로 만들어져 있어 게임을 느리게 합니다.
  • 크리에이터 평판: 평점과 제작자를 확인하면 품질을 어느 정도 거를 수 있습니다.

외부 3D 에셋 임포트 (Import 3D)

블렌더 등에서 만든 FBX/OBJ 파일은 Home 탭 > Import 3D로 가져옵니다. MeshPart로 변환되어 Workspace에 추가됩니다. 크기가 이상하면 임포트 창에서 Scale을 조절하세요.

RBX

이미지(데칼, 텍스처)를 쓰려면 create.roblox.com의 Creator Hub에서 업로드한 뒤 발급된 Asset ID를 rbxassetid://숫자 형식으로 입력합니다.

플러그인 — Studio에 기능 추가하기

플러그인은 Studio에 새 도구를 더해주는 확장 프로그램입니다. 에셋이 "가져다 쓰는 물건"이라면, 플러그인은 "작업을 도와주는 도구"입니다. 손으로 하면 한참 걸리는 일을 버튼 한 번으로 끝내주는 것들이 많아 맵 제작 속도를 크게 올려줍니다.

  1. 찾기: 상단 Plugins 탭 > Creator Store (또는 create.roblox.com의 Creator Store > Plugins).
  2. 설치: 원하는 플러그인을 Install (무료/유료가 있습니다). 설치하면 Plugins 탭에 버튼이 생깁니다.
  3. 관리: Plugins 탭 > Manage Plugins에서 켜고 끄거나 업데이트.
!!

플러그인도 Toolbox 모델처럼 신뢰가 중요합니다. 설치 수가 많고 평판이 좋은 공식/유명 제작자의 것을 쓰세요. 플러그인은 Studio 안에서 코드를 실행할 수 있으므로, 출처가 불분명한 것은 피하는 게 안전합니다.

구체적으로 어떤 플러그인이 맵 제작에 유용한지는 중급(건축 보조)과 고급(최적화)에서 실제 작업과 함께 다룹니다.

08머티리얼과 색 초급

같은 파트도 머티리얼과 색에 따라 완전히 다른 맵이 됩니다. 맵의 첫인상을 결정하는 요소.

머티리얼 종류

머티리얼느낌맵 활용
Plastic / SmoothPlastic매끈, 무광기본값. 로우폴리 스타일 맵의 표준
Wood / WoodPlanks나무 결집, 다리, 울타리, 부두
Brick / Cobblestone벽돌, 자갈건물 벽, 길바닥, 성벽
Concrete / Slate거친 돌도시 바닥, 절벽, 바위
Metal / DiamondPlate / CorrodedMetal금속기계, 공장, 폐허
Grass / Sand / Ground자연 표면야외 지면 파트
Neon자체 발광간판, 마법 효과, 용암 코어, 빛 장식
Glass / ForceField투명 계열창문, 보호막, 물 표현
Ice / Snow얼음, 눈설원 맵
TIP

Neon은 Lighting의 Bloom과 만나면 진짜 빛처럼 번집니다. 어두운 맵에서 Neon 파트 하나만 잘 써도 분위기가 살아납니다.

색 정하기

  1. 파트 선택 > Properties > Color 클릭 > 색상표에서 선택.
  2. 또는 Home 탭의 Color 버튼으로 빠르게 칠하기.
  3. 정확한 색이 필요하면 색상표 아래 RGB 값을 직접 입력합니다.

맵 색 조합 기본기

  • 3색 규칙: 주색(바닥/벽 70%) + 보조색(소품 25%) + 포인트색(5%)로 제한하면 정돈돼 보입니다.
  • 채도 낮추기: 원색(빨강 255,0,0)보다 살짝 탁한 색이 배경으로 편안합니다. 포인트에만 쨍한 색.
  • 같은 색 계열로 명암: 벽은 밝은 회색, 그림자 지는 부분은 어두운 회색 — 같은 계열로 단계만 주면 자연스럽습니다.
  • 참고: 좋아하는 게임 맵 스크린샷을 옆에 띄워두고 색을 따라 잡는 게 가장 빠른 연습입니다.

Transparency와 Reflectance

  • Transparency (0~1): 투명도. 유리창 0.5, 거의 안 보이는 가이드 0.9.
  • Reflectance (0~1): 하늘 반사. 금속이나 물 느낌에 0.1~0.3 정도만 — 과하면 어색합니다.

09테스트와 퍼블리시 초급

만든 맵을 테스트하고 세상에 공개하는 방법.

테스트 방법

방법단축키용도
PlayF5스폰 지점에서 캐릭터로 플레이
Play Here-현재 카메라 위치에서 플레이
RunF8캐릭터 없이 서버만 실행. 물리/스크립트 확인
StopShift+F5테스트 종료
!!

Play 중에 한 변경은 Stop을 누르면 사라집니다. Play 중 좋은 위치/값을 찾았다면 메모하거나, 변경한 오브젝트를 우클릭 > Copy 하고 Stop 후 Paste 하세요.

저장과 퍼블리시

  1. 로컬 저장: File > Save to File. 내 컴퓨터에 .rbxl 파일로 저장.
  2. 클라우드 저장: File > Save to Roblox. 계정에 저장되어 어디서든 열 수 있음.
  3. 퍼블리시: File > Publish to Roblox (Alt+P). 이름, 설명, 장르 입력.
  4. 퍼블리시 후 Game Settings > Permissions에서 Public으로 바꿔야 다른 사람이 플레이할 수 있습니다.
